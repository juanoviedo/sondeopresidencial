¿Cómo funciona el sistema de votación?

Para participar en el sondeo, solo debes enviar un mensaje por WhatsApp al número oficial del sondeo. Nuestro sistema lee automáticamente tu último mensaje y detecta palabras clave asociadas a cada candidato presidencial.

Solo necesitas enviar el nombre y/o apellido del candidato para que tu voto sea registrado.

🗳 Votar ahora por WhatsApp

Por ejemplo:
• Si escribes "Iván", tu voto contará para Iván Cepeda.
• Si escribes "Esprilla", tu voto contará para Abelardo de la Espriella.
Y así sucesivamente con cada candidato.

Además, el sistema también contempla posibles errores comunes al escribir los nombres. Sabemos que algunas personas pueden cometer errores ortográficos, escribir solo una parte del nombre o incluir pequeñas variaciones al momento de votar.

Nuestro sistema identifica estas variaciones y analiza el mensaje de forma inteligente para determinar con precisión la intención del votante. Es decir, incluso si el nombre contiene un error leve o está incompleto, el voto será asignado correctamente al candidato correspondiente (en la mayoria de los casos).

De esta manera garantizamos que tu participación sea válida, incluso si hay pequeños errores al escribir.

Seguridad y Protección de Datos

Este sondeo ha sido diseñado priorizando la privacidad y la seguridad de los participantes. El sistema no publica ni almacena números de teléfono en la visualización de Power BI.

La base de datos visible únicamente contiene un identificador interno (ID anónimo) generado automáticamente en el momento en que se recibe el mensaje. Este ID no permite identificar al votante ni acceder a su número celular.

En el improbable caso de un acceso no autorizado a la visualización pública, la única información visible serían registros anónimos sin datos personales. No se comparten números de teléfono, nombres ni información privada.

Este proyecto es una iniciativa independiente y sin afiliación política. Nuestro único objetivo es realizar un sondeo ciudadano y demostrar cómo la tecnología puede utilizarse para modernizar y hacer más eficientes los procesos participativos en el país.

Creemos en el uso responsable de la tecnología como herramienta para fortalecer la transparencia, la participación y el desarrollo institucional.

¿Cómo garantizamos que cada persona vote una sola vez?

El sistema está diseñado para asegurar que cada participante tenga un único voto válido. Para lograrlo, utilizamos un identificador interno asociado a cada participante.

Cuando una persona envía un mensaje, el sistema verifica automáticamente si ya existe un registro previo asociado a ese identificador. En caso de existir, el nuevo voto reemplaza el anterior, manteniendo siempre un único voto válido por participante.

Esto significa que:
• No es posible acumular múltiples votos desde el mismo participante.
• Siempre se conserva únicamente el último voto válido enviado.
• El sistema evita duplicidades de manera automática.

Actualmente utilizamos una modalidad estándar de Power BI que establece límites en la cantidad de actualizaciones automáticas permitidas por día. Esto se debe a políticas de optimización de recursos del servicio, ya que las actualizaciones en tiempo real o con alta frecuencia requieren capacidades premium orientadas a entornos empresariales.

Debido a estas restricciones técnicas, el tablero no se actualiza en tiempo real, sino en intervalos programados a lo largo del día. Esto nos permite mantener el proyecto de forma sostenible mientras garantizamos estabilidad y buen rendimiento de la plataforma.

¿Por qué no respondemos los mensajes de WhatsApp?

Este sistema está diseñado exclusivamente para la recepción y procesamiento automático de votos. Por esta razón, no enviamos respuestas automáticas ni mensajes de confirmación a los participantes.

Actualmente utilizamos un mecanismo de automatización (RPA) únicamente para la lectura y validación de los mensajes entrantes. Implementar respuestas automáticas mediante este mismo método podría generar restricciones o bloqueos por parte de la plataforma de mensajería, ya que WhatsApp limita el uso automatizado no oficial.

Existe la alternativa de utilizar la API oficial de WhatsApp Business para enviar respuestas automatizadas. Sin embargo, este tipo de integración implica costos operativos elevados, orientados principalmente a soluciones empresariales de gran escala.

Con el fin de mantener este proyecto como una iniciativa independiente, sostenible y de acceso abierto, hemos optado por no implementar respuestas automáticas.

Si tu mensaje contiene correctamente el nombre o palabra clave del candidato, puedes tener la tranquilidad de que tu voto será procesado y reflejado en la siguiente actualización programada del tablero.

¿Por qué desarrollamos este sistema?

Este proyecto nace como una iniciativa tecnológica independiente cuyo propósito principal es demostrar cómo las herramientas digitales pueden aplicarse a procesos participativos y gubernamentales de manera eficiente, transparente y auditable.

A lo largo de la historia, los Estados que no modernizan sus procesos mediante tecnología tienden a enfrentar problemas de ineficiencia operativa, falta de control, altos costos administrativos y mayores riesgos de corrupción. La transformación digital no es únicamente una tendencia global: es una necesidad estructural para garantizar instituciones más sólidas y confiables.

Este sondeo no pretende reemplazar los mecanismos oficiales de votación. Sin embargo, funciona como un ejercicio demostrativo para analizar cómo podrían diseñarse sistemas electorales digitales más seguros, trazables y auditables.

El potencial de las elecciones digitales

En algunos esquemas tradicionales de clientelismo electoral documentados en distintos contextos, la movilización de votantes no ocurre de manera espontánea, sino mediante redes organizadas. A determinados intermediarios locales se les asigna la tarea de reunir un número específico de personas que se comprometan a apoyar a un candidato. A cambio, pueden existir incentivos económicos, ayudas puntuales o promesas de beneficios futuros.

Para organizar esa movilización, estas redes suelen recopilar información básica de los votantes, como su número de identificación, con el fin de consultar en los portales oficiales cuál es su puesto y mesa de votación asignados. De esta manera, quienes coordinan la estructura conocen con anticipación en qué mesas debería verse reflejado el apoyo prometido.

Posteriormente, al publicarse las actas oficiales de escrutinio de cada mesa, es posible comparar los resultados obtenidos en esas ubicaciones específicas con las zonas donde se concentró la movilización. Aunque el voto es secreto y no puede verificarse individualmente, el análisis agregado por mesa permite hacer inferencias estadísticas sobre el cumplimiento de los compromisos previos. Precisamente este tipo de dinámicas, basadas en intermediación territorial y consolidaciones manuales, es lo que muchos expertos consideran que podría mitigarse mediante sistemas digitales más trazables, auditables y estructuralmente menos dependientes de controles físicos.

Al eliminar procesos manuales y trasladar la validación a sistemas criptográficos y registros digitales inmutables, se reduce el margen de error humano y se dificulta la interferencia indebida en los resultados.

Además, un sistema digital permitiría que múltiples entidades independientes — universidades, organismos técnicos nacionales e incluso observadores internacionales— puedan auditar en tiempo real la integridad del proceso. La trazabilidad digital facilita la verificación matemática de los resultados, algo que en sistemas físicos depende de revisiones manuales posteriores.

La tecnología bien implementada puede fortalecer la transparencia, reducir costos operativos, aumentar la eficiencia institucional y elevar la confianza ciudadana en los procesos democráticos.

Este proyecto busca abrir la conversación sobre cómo Colombia y otros países pueden avanzar hacia modelos más modernos, auditables y tecnológicamente robustos en sus procesos estatales, siempre bajo marcos legales adecuados, supervisión institucional y estándares internacionales de seguridad.

Creemos que el futuro de la gobernanza incluye sistemas digitales transparentes, verificables y diseñados con altos estándares técnicos, donde la tecnología actúe como herramienta de fortalecimiento democrático.

Sobre el uso de los datos

Este proyecto no tiene como objetivo la explotación comercial de información. Los datos gestionados dentro del sistema se utilizan exclusivamente para garantizar el correcto funcionamiento del sondeo, particularmente para asegurar que cada participante tenga un único voto válido.

En ningún caso se utilizan los registros para campañas publicitarias, envío de promociones, cesión a terceros ni segmentación comercial. De hecho, desde una perspectiva técnica y estratégica, el uso de bases de datos de este tipo para publicidad sería ineficiente frente a los canales formales y especializados que existen hoy en día, como las plataformas oficiales de anuncios digitales.

La información administrada cumple una función estrictamente operativa: evitar duplicidad de votos y mantener la integridad del ejercicio. No se realizan análisis de comportamiento, perfiles individuales ni ningún tipo de explotación secundaria de los registros.

Este es un ejercicio independiente con fines demostrativos y académicos. Nuestro interés está centrado en explorar cómo los sistemas digitales pueden fortalecer procesos participativos y mejorar la eficiencia institucional, no en la monetización de datos.

Creemos que la tecnología aplicada al sector público debe construirse bajo principios de proporcionalidad, minimización de datos y uso responsable. Esa misma filosofía guía el desarrollo de este proyecto.
